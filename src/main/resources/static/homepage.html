IndeedBot API
job listings scanner by AdamWandoch

Description

This is an experimental project to explore Java ecosystem of technologies for web development.

The application retrieves html from HERE and parses it creating a list of IndeedJob objects containing information about the job title, company name, unique "indeed job id" and a link to view the listing. All data is persisted to Postgres Database hosted on AWS RDS and retrieved on application restart. Data is exposed through a number of basic endpoints in JSON format.

This Spring Boot powered RESTful API is running on Heroku and another instance on AWS, and it has a task that hits the custom "/ping" endpoint in a specified time interval to keep itself awake. Another scheduled task scans Indeed.ie and updates the job list periodically.
