IndeedBot
job listings scanner by AdamWandoch

Description

This is an experimental project to explore Java ecosystem of technologies for web development.

The application retrieves html from HERE and parses it creating a list of IndeedJob objects containing information about the job title, company name, unique "indeed job id" and a link to view the listing. All data is persisted to Postgres Database running on Heroku and retrieved on application restart. Data is exposed through a number of basic endpoints in JSON format.

This Spring Boot powered RESTful API is running on a free Heroku dyno which could sleep occasionally (give it a few seconds to respond); it has however a scheduled task that hits the custom "/ping" endpoint every 10 minutes to keep itself awake. Another scheduled task scans Indeed.ie and updates the job list periodically.

    Available endpoints:

  • /jobs (returns a list of all job resources) GO!
  • /job/{index} (returns a single job resource) GO!
  • /jobs/title/{keyword} (returns a list of jobs containing the keyword in title) GO!
  • /jobs/company/{keyword} (returns a list of jobs containing the keyword in company name) GO!
