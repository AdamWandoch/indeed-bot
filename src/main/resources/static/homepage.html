IndeedBot
job listings scanner by AdamWandoch

Description

This is an experimental project to explore Java ecosystem of technologies for web development.

The application retrieves html from HERE and parses it creating a list of IndeedJob objects containing information about the job title, company name, unique "indeed job id" and a link to view the listing. All data is persisted to Postgres Database hosted on HEROKU and is available in JSON format through indeed-bot-api.
